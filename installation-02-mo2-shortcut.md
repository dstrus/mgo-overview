# Mod Organizer 2

As you may know, you can manage the list and launch the game via Mod Organizer 2.

Here I have a nearly-pristine installation of MGO SFW 3.8.8.1. I’ve only added a couple of mods that I know will be included in the next release and that will make this video look quite a bit better.

Personally, I play several mod lists for a couple different games that all use Mod Organizer 2 (which, for the sake of brevity, I’ll call “MO2” from now on). Now you can pin each copy of MO2 to the Taskbar or Start Menu, but they’ll all have the same icon, and they’ll all be labeled `Mod Organizer`. This isn’t a “how to use Windows” tutorial, but frankly they’ve made this sort of customization such a pain recently that I’ll show you how I’ve done it. Remember, you can always skip ahead.

## Pinning a shortcut to Mod Organizer

Find your MGO installation folder in Windows Explorer, and find `ModOrganizer.exe`. Right-click the icon and drag it just a bit, within the same folder, and choose `Create a shortcut`. You can name the shortcut whatever you want, like `Skyrim VR MGO`. You can also change the icon. Right-click, Properties, Change Icon. If nothing else, you can set it to Skyrim’s icon. I think the Skyrim VR icon is ugly, so how about the Skyrim SE icon? Or a custom icon for MGO!

Having done that, I'll right-click an existing pinned shortcut and choose `Show Original`. This doesn't actually show the _original_ original, but rather the pinned shortcut. It's weird. Anyway, drag your MGO shortcut into the same location as the other, and right-click, and pin it to the Start Menu.

## Launching Mod Organizer

Once you launch it, MO2 takes a little while to start. When it has finished loading, you can launch the actual game with the `▶️ Run` button, with `Play Game (SKSE)` selected in the nearby dropdown.

There are other tools in this dropdown that you may need on occasion if you start fiddling with the mod list, but we’re not there yet. Also, you can create a shortcut that actually launches the game instead of launching MO2. If you’d rather pin that to the Start Menu or Taskbar, now you know how.
