# Launching Mad God's Overhaul

Let’s go ahead and launch the game, and we’ll circle back to some of the initial configuration in a bit. In Mod Organizer, click the `▶️ Run` button with `Play Game (SKSE)` selected in the nearby dropdown, near the upper-right of your MO2 window. If the dropdown is set to something other than `Play Game`, change it to `Play Game (SKSE)`. There are other tools in this dropdown that you may need on occasion if you start fiddling with the mod list, but we’re not there yet. Also, you can create a shortcut that actually launches the game instead of launching MO2.

## Compiling Shaders

When you launch the game for the first time, you should see “Compiling Shaders” in the upper-left corner of the desktop view. Compiling the shaders may take several minutes, but this is a one-time thing. Once these shaders are compiled, they will be saved to disk, and you won’t have to sit through this anymore.

This is part of a bunch of mods collectively known as [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492), which allows for a number of advanced graphical features that were not in the original game. It’s a big part of why MGO looks so much better than vanilla, even aside from all the improved assets. Although Community Shaders does support VR out of the box, MGO uses an [unofficial, custom version of Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/166950) by Troned that supports some features that don’t work in VR in the official release.

## Community Shaders overlay

While the shaders are compiling, you can adjust Community Shaders’ settings if you like. You can bring up the CS overlay by pressing the `⌨️ END` key on your keyboard with the game window focused. Pressing `⌨️ END` again will close the overlay. Getting deep into these settings is beyond the scope of this video, but this is how you get to the settings, and now you know that Community Shaders is the name of the mod behind the advanced graphical features. So can you read up on how to use it elsewhere. Before I close the overlay, however, I want to point out the tiny icon for saving your changes. If you don’t save, you will lose any changes you’ve made the next time you start the game.

## Main Menu

Once you’re past that, some differences from vanilla Skyrim VR are noticeable immediately. Unmodded Skyrim VR loads very quickly, and the main menu appears in a scenic area with a waterfall. This area is just for the main menu, by the way. It’s not a location in the game itself.

With MGO, it takes a bit to reach the main menu, even if the shaders are already compiled. While you wait, you’ll see the Seal of Akatosh floating in a black void surrounded by particle effects… beneath where the ground would be. Once you reach the main menu, it will be straight in front of you, and it’s basically the menu from the flatscreen version of the game. The only settings you can change from this settings menu have to do with controllers. You can change between motion controllers and a traditional controller, though I’m not sure how well traditional controllers work with MGO, or if they work at all. You can also set your primary motion controller to either the left or the right.
