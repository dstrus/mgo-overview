# MGO Overview scripts, transcripts, and written tutorials

This is a written companion to a video (or video series... time will tell) providing an overview of the Mad God's Overhaul mod list for Skyrim VR.

For now, this is just the Markdown files as I wrote them (or, in places, as I pasted them from Google Docs and then reformatted them for Markdown), but I'm likely to turn this into a portal of sorts later.

In the interest of proving this project is not vaporware, I've posted a [demo video](https://www.youtube.com/watch?v=uhw6HE2tDa8) with a portion of the intro.

The MGO footage in that clip is almost certain to be replaced with higher-quality footage, and I'll probably replace "From Past to Present" with a more exciting track. I'll also likely reduce some of the noise in the narration.

Again, this is just to prove that it's not vaporware.
